# 印象·葵潭

一个展示广东省揭阳市惠来县葵潭镇的旅游网站，包含景点介绍、美食特产、文化民俗和旅游攻略等内容。

**在线预览：https://arousx.github.io/travel-website/**

## 项目结构

```
├── index.html          # 首页
├── attractions.html    # 景点介绍
├── food.html           # 美食特产
├── culture.html        # 文化民俗
├── travel.html         # 旅游攻略
├── css/
│   └── css.css         # 样式文件
├── JavaScript/
│   └── js.js           # 轮播图脚本
└── img/                # 图片资源
```

## 主要功能

- 首页轮播图展示
- 美食推荐卡片
- 景点推荐卡片
- 响应式导航栏
- 页脚信息展示

## 本地运行

直接在浏览器中打开 `index.html` 即可查看网站。
